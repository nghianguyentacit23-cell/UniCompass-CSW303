# UniCompass - AI Academic Copilot

UniCompass is a CSW303 Software Engineering group project. The product helps university students make clearer academic decisions, especially around semester planning, course registration, possible study risks, and GPA goals.

## Week 1 Scope

Week 1 focuses on Requirements Engineering:

1. Product Vision
2. 2 Personas
3. 3 Scenarios
4. Trello board setup evidence
5. Team Operating Agreement with RACI table

The Week 1 submission materials are stored in:

```text
week1-submission/
```
